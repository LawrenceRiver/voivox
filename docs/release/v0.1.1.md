# Voice Vac v0.1.1 — live Chrome capture fix

This patch release is the first build validated through a real Chrome `tabCapture` run against a playing Xiaohongshu video.

## Fixed

- Route Offscreen Document state through the service worker because Chrome does not expose `chrome.storage` in offscreen contexts.
- Prevent a startup deadlock by allowing offscreen state reads/writes to bypass the serialized user-command queue.
- Keep compact q8 Whisper on reliable bundled ONNX Runtime WASM instead of experimental q8 WebGPU execution.
- Use bounded greedy decoding with incomplete-sequence and repeat suppression so noisy clips cannot produce runaway repeated output.

## Verified

- Real active-tab capture entered recording, downloaded/loaded the pinned Quality model, completed local inference, and displayed finite text without a speech API.
- Chrome extension reliability, type, build, packaging, and repository-wide test suites are run before tagging.
- The README now includes the real completed popup screenshot and a linked acceptance record.
